# Assets

This folder holds public launch assets for the Mantle repository.

## Structure

- `brand/` — reusable brand assets such as the app icon
- `images/` — static screenshots or placeholders used in README
- `gifs/` — short workflow recordings
- `diagrams/` — architecture and product diagrams
- `social/` — comparison and social share graphics

Placeholder assets are included so the public repo has stable paths before final media is recorded.
The default public brand icon is `brand/mantle-app-icon.svg`, aligned with the current `Anchored Loop` direction.
